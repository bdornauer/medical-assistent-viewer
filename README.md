## Available Scripts

In the project directory, you can run:

### `npm install`
To load the packages.  

### `npm run build`
Builds the project. 

### `npm run start`
Executes the application.\

### `npm run dev`
Executes the application in development mode.\

### `npm run heroku-postbuild`
Equal to `npm run build` but for Heroku-Service\
